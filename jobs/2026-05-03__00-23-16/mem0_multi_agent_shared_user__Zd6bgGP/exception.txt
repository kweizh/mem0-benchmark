## Background
In multi-agent systems, agents often share user context but maintain separate "Agent Memories" (e.g., personality, instructions). Mem0 isolates memories by entity IDs (`user_id`, `agent_id`, etc.). To retrieve both a user's memory and a specific agent's memory for a prompt, you must query for both correctly.

1. Initializes a Mem0 `Memory` instance using the open-source configuration (default providers are fine).

   - User memory (`user_id="alice"`): "I am a vegetarian and allergic to nuts."
   - Agent memory (`agent_id="coach"`): "You are a fitness coach. Always encourage the user to push their limits."
   - Agent memory (`agent_id="nutribot"`): "You are a nutritionist. Recommend high-protein plant-based meals."
