# Delete All Memories for a Specific Agent in Mem0

## Background
Mem0 is a memory layer for AI agents. Memories can be scoped to specific entities like `user_id`, `agent_id`, `app_id`, and `run_id`. In a multi-agent system, you might want to clear all memories associated with a specific agent without affecting other agents or users.

## Requirements
- Initialize a Mem0 Open Source `Memory` instance using the default configuration (which uses Qdrant and OpenAI by default, or just the default in-memory/local setup if no config is provided, but we will provide a basic config).
- Add a few memories for `agent_id="travel-agent"`.
- Add a few memories for `agent_id="support-agent"`.
- Delete all memories associated with `agent_id="travel-agent"`.
- Verify that memories for `agent_id="travel-agent"` are deleted, while memories for `agent_id="support-agent"` remain intact.
- Save the count of remaining memories for `agent_id="support-agent"` to a log file.

## Implementation Guide
1. Create a Python script `/home/user/mem0-project/delete_memories.py`.
2. Initialize `from mem0 import Memory` and `memory = Memory()`.
3. Add memories: 
   - `memory.add("I like window seats", agent_id="travel-agent")`
   - `memory.add("User prefers email support", agent_id="support-agent")`
4. Delete all memories for the travel agent: `memory.delete_all(agent_id="travel-agent")`.
5. Get all memories for the support agent: `support_mems = memory.get_all(agent_id="support-agent")`.
6. Write the number of remaining support agent memories to `/home/user/mem0-project/output.log`.

## Constraints
- Project path: `/home/user/mem0-project`
- Log file: `/home/user/mem0-project/output.log`
- Use Mem0 Open Source version.
