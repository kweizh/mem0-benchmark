# Mem0 Search with Limit

## Background
Mem0 manages long-term memory for AI agents. When retrieving context, you often want to limit the number of results returned to fit within an LLM's context window.

## Requirements
- Write a Python script `/home/user/project/search.py`.
- Initialize a Mem0 `Memory` instance (Open Source version).
- Add the following 4 facts for `user_id="alice"`:
  1. "Alice loves to play tennis."
  2. "Alice is allergic to peanuts."
  3. "Alice works as a software engineer."
  4. "Alice has a dog named Max."
- Search for the query "What sports does Alice play?" for `user_id="alice"`, but limit the search to return exactly 2 results.
- Write the search results (a list of dictionaries) to `/home/user/project/results.json`.

## Implementation Guide
1. Use `from mem0 import Memory` and `m = Memory()` to initialize the default open-source memory.
2. Use `m.add()` to add the facts.
3. Use `m.search(query="...", user_id="alice", limit=2)` to search with a limit.
4. Dump the results to `results.json` using the `json` module.
5. Run the script to generate the output.
